# CR常総学院 〜青春御伽769ver.〜

『常総学院シミュレーター』のキャラクター・背景・BGM/SE素材を使用した、
ブラウザで動作するパチンコ実機シミュレーターです。

## 遊び方

ローカルサーバーを起動してブラウザで開きます（Node/Python不要）:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File serve.ps1
# → http://localhost:8931/ を開く
```

- **ハンドル（スライダー）** を回すと玉が発射されます（100発/分）
- 左〜中程度でヘソ（START）を狙う「左打ち」、右端（赤ゾーン）で「右打ち」
- キーボード: `←` `→` で微調整、`Space` で左打ち⇔右打ち切り替え
- 大当り中・RUSH中・チャンスタイム中は **右打ち** で電チュー＆アタッカーを狙う

## 構成

```
index.html        エントリーポイント

js/spec.js        スペック・キャラ・リーチ定義

js/board.js       盤面物理演算（釘・レール・玉・入賞口）
js/machine.js     ゲーム進行（抽選・保留・モード管理）
js/main.js        起動・UIバインド
js/pegs_data.js   筐体アートから自動検出した釘の当たり判定データ
assets/           画像・音声素材（常総学院シミュレーターほか）
imagin/           筐体画像（液晶透過版）・発光差分・玉スプライト
serve.ps1         簡易ローカルサーバー（PowerShell）
```

## 筐体との一体化について

- 筐体画像は `筐体点灯なし_液晶透過版.PNG` を使用し、透過部分の**奥**に液晶（映像レイヤー）を配置
- レイヤー構成: 液晶（奥）→ 筐体画像 → 玉・入賞口キャンバス → 発光差分（手前）
- 筐体アートに描かれた金色の釘を画像解析で検出（262本）し、当たり判定として使用。玉は絵の釘の間を通って落ちる
- 「始動」「入賞口」「右打ち」プレートの位置がそのままヘソ・アタッカー・右打ちレーンに対応
- 発光演出は「点灯あり−点灯なし」の画像差分を大当り・RUSH時にオーバーレイ
